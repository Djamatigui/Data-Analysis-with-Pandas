# 🌍 Global Fuel Prices Analysis (2020 - 2026)

## 📌 Description
Ce projet est une analyse des prix du carburant dans différents pays entre 2020 et 2026.

Il a été réalisé avec **Python et Pandas** pour explorer, nettoyer et analyser les données.

---

## 🎯 Objectifs
- Explorer les prix du carburant par pays
- Analyser l’évolution des prix dans le temps
- Comparer les pays entre eux
- Identifier les pays les plus chers et les plus stables
- Étudier la volatilité des prix

---

## 🛠️ Technologies utilisées
- Python 🐍
- Pandas

---

## 📊 Étapes de l’analyse
- Chargement des données avec Pandas
- Exploration des données (head, info, describe)
- Nettoyage des données (valeurs manquantes, formats)
- Analyse descriptive (moyenne, min, max)
- Analyse par pays
- Analyse temporelle
- Comparaison entre pays

---

## 📈 Résultats
- Les prix varient fortement selon les pays
- Certains pays ont des prix plus stables que d’autres
- On observe des différences importantes sur la période 2020 - 2026

---

## 💡 Conclusion
Cette analyse permet de mieux comprendre les différences de prix du carburant dans le monde et leur évolution dans le temps en utilisant uniquement Pandas.

---
